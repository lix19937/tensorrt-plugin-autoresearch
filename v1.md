assume has install `Claude Code`   

1. start claude  
```bash
cd tensorrt-plugin-autoresearch   && bash  ./claude-auto.sh
```

2. choose a skill  
```
/trt-plugin-kernel-latency-auto-opt
```

If no error, now enter `auto-tune`
